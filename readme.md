---

## ✨ Core Highlights

-   **3D Hero Experience:** Interactive mouse-parallax 3D environment with fluid spring physics.
-   **Nebula Guide AI:** Integrated bilingual (Bangla/English) assistant powered by **Groq + Llama 3.3**.
-   **Skill Orbits:** Dynamic "Bubble Pop" skill showcase using original brand icons and zero-gravity floating animations.
-   **Project Case Studies:** 6 premium full-stack and 3D projects with detailed documentation on challenges and future improvements.
-   **Ultra-Thin UI:** Glassmorphic, minimal design language optimized for both Dark and Light modes.

## ⚙️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/niloyhakimai/portfolio-niloy.git](https://github.com/niloyhakimai/portfolio-niloy.git)
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Setup Environment Variables:**
    Create a `.env.local` file and add your keys:
    ```env
    GROQ_API_KEY=your_key_here
    NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_key_here
    DATABASE_URL=your_postgresql_url
    ```

4.  **Run Development Server:**
    ```bash
    npm run dev
    ```

## 🎨 Design Philosophy

This portfolio follows a **"Space Floating"** aesthetic. It prioritizes depth through z-index layering, frosted glass textures, and subtle neon glows. Every interaction is calculated to feel tactile—responding to the user's cursor as if every element is floating in a zero-gravity digital environment.
